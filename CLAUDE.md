# WindowMark — Claude Instructions

macOS menu bar app (Swift 6, XcodeGen, zero third-party deps) for marking windows and cycling through them with global hotkeys. Build: `xcodegen generate && xcodebuild -scheme WindowMark -configuration Debug build`
